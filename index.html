Overview Abuse Threat Best Practices Deep Dive

ภัยคุกคามในโลกออนไลน์มีอยู่รอบตัวเรา มาเรียนรู้วิธีป้องกันและรับมือ เพื่อให้คุณท่องโลกดิจิทัลได้อย่างมั่นใจและปลอดภัย

การล่วงละเมิดทางไซเบอร์ (Cyber Abuse)

พฤติกรรมที่เป็นอันตราย หรือสร้างความเสียหายต่อผู้อื่นผ่านโลกออนไลน์

Cyberbullying

การกลั่นแกล้ง ให้ร้าย ด่าว่า หรือรังแกผู้อื่น

Cyber Harassment

การล่วงละเมิดที่รุนแรงและต่อเนื่องกว่า

Cyberstalking

การสอดแนม ติดตามการเคลื่อนไหวซ้ำๆ

Trolling

การจงใจโพสต์ยั่วยุให้เกิดความขัดแย้ง

ภัยคุกคามทางไซเบอร์ (Cyber Threat)

การกระทำโดยมิชอบที่มุ่งร้ายต่อข้อมูลหรือระบบคอมพิวเตอร์ของคุณ

Phishing

การหลอกลวงให้คลิกลิงก์หรือกรอกข้อมูลส่วนตัว

Malware

โปรแกรมที่ออกแบบมาเพื่อสร้างความเสียหาย

ภัยชอปปิงออนไลน์

การหลอกลวงให้เสียทรัพย์สินจากการซื้อของ

Ransomware

มัลแวร์ที่เข้ารหัสไฟล์เพื่อเรียกค่าไถ่

ภัยจากแอปแชต

การสวมรอยเป็นคนรู้จักเพื่อหลอกยืมเงิน

แนวทางปฏิบัติเพื่อความปลอดภัย (Security Best Practices)

เกราะป้องกันพื้นฐานที่คุณสร้างได้เองตั้งแต่วันนี้

Strong Passwords & 2FA

การตั้งรหัสผ่านที่รัดกุมและเปิดใช้งาน 2FA

Data Backup

การป้องกันข้อมูลสูญหายจากเหตุไม่คาดฝัน

Software Updates

การปิดช่องโหว่ความปลอดภัยด้วยการอัปเดต

Public Wi-Fi Security

ความเสี่ยงจากการใช้เครือข่ายไร้สายสาธารณะ

ความเข้าใจเชิงลึก (Developer Deep Dive)

แนวคิดทางเทคนิคสำหรับนักพัฒนา: Hashing, Encoding, และ Encryption

Hashing (การแฮช)

การสร้างลายนิ้วมือดิจิทัล (Integrity)

Encoding (การเข้ารหัส)

การแปลงข้อมูลเพื่อความเข้ากันได้ (Compatibility)

Encryption (การเข้ารหัสลับ)

การปกป้องความลับของข้อมูล (Confidentiality)
